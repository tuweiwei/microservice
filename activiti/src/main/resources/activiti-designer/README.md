# activiti-designer

#### 介绍
基于bpmnjs 的 activiti 模型设计器

#### 软件架构
软件架构说明


#### 安装教程

1.  切换到项目目录下运行  npm install 安装依赖
2.   运行 npm run dev 启动

#### 使用说明

1. 可在页面进行拖拽设计activiti的流程图，支持导出bpmn流程图和svg图片
